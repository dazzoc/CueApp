> # Cue Music App
> 
A simple web application that returns color information baseded apon the user inputing a Hex code. A good way to practice 0-F Hex code and learn more information on the color.
___
## Technologies Used
- Python
- Django
- PostgreSQL
- HTML5
- CSS3
- JavaScript
- Materialize
- Google Fonts
