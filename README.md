> # Cue Music App
> 
A web application where users can track their music. Users can add the albums they are listening to, add the date they listened to it on and witch device they used to listen to the album.
___
## Technologies Used
- Python
- Django
- PostgreSQL
- HTML5
- CSS3
- JavaScript
- Materialize
- Google Fonts
- Font Awesome Icons
